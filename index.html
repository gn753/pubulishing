Responsive Web 개인 포트폴리오 페이지

반응형 웹을 지원하는 개인 포트폴리오 페이지입니다. 많은 작업물을 쉽게 볼 수 있도록 제작하였고 유지보수가 쉽도록 코딩에 신경썼습니다. 탭 메뉴/라이트 박스/모바일 내비게이션에 신경써서 제작하였습니다.

  • 제작페이지 :1개
  • 제작방식 :html + css + jquery
  • 플러그인 : UIKIT(라이트박스)
  • UI 디자인 : PhotoShop
  • 제작 참여율 : 100%
모바일

Desktop 취업성공패키지

공기관 사이트를 가상으로 리뉴얼 작업했습니다. 데스크탑 버전을 지원하며 메인페이지와 사업소개, 자가진단, 참여방법, 약관동의 등 7개 페이지를 제작하여 기획 당시 다양한 형식의 웹페이지를 선보일 수 있도록 목표 한 프로젝트입니다. 전체 폰트는 Noto Sans Kr을 사용했습니다
유기적으로 연결되어 있는 여러 웹페이지를 제작하면서 코드가 부딪히지 않도록 짜임새 있게 코딩해야한다는 것을 배웠습니다.

  • 제작페이지 :7개
  • 제작방식 : html + css + jquery
  • 플러그인 : xeicon(아이콘 웹폰트)
  • UI 디자인 : PhotoShop
  • 제작 참여율 : 100%

보유스킬

HTML

웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱이 가능합니다.

CSS

안정적인 CSS속성을 사용해 HTML Element를 배치하고 디자인합니다. 특별한 경우 CSS3에서 새롭게 지원하는 속성을 활용해 좀 더 다양한 CSS 배치와 디자인을 하고 있습니다.

JQUERY

네비게이션, 탭 메뉴, 어코디언 등 웹사이트 상호작용에 필수적인 요소들은 플러그인을 사용하지 않고 순수 제이쿼리 코딩으로 제작합니다.

Responsive Web

크로스브라우징(Cross Browsing)을 위해 안정적인 기존 방식인 float, position, Media Query를 활용한 반응형 웹 제작 경험을 충분히 가지고 있습니다. 필요에 따라 Flex방식으로 반응형 웹 사이트 제작도 충분히 가능합니다

Mobile Web App

모바일 우선을 위해 Mobile First에 준하는 모바일 웹 앱 제작경험도 충분히 가지고 있습니다. UI/UX 측면에서 미적인 디자인뿐만 아니라 모바일 사용자 경험에 최적화를 위해 효율적인 컨텐츠 배치를 우선적으로 생각합니다

PhotoShop

웹사이트 UI 디자인 작업, 배너 디자인을 위한 포토샵 사용에 익숙합니다
