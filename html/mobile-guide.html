MOBILE WEB APP PORTFOLIO

웹소설 플랫폼 소설 네트워크에 모바일 웹앱을 제작 해 사용자의 편의성과 레이아웃을 중점으로 레이아웃을 제작했습니다

LOGIN

로그인은 그 브랜드의 고유 대표색상을 사용하여 좀 더 사용자가 직관적으로 알 수 있게 사용하였습니다

MENU

사용자가 정보를 좀 더 직관적으로 전달하기 위해 메뉴를 간략하게 구성하였고 이중터치가 되지 않도록 알맞는 크기로 메뉴의 사이즈를 구성하였습니다.

BEST

소설네트워크의 대표인 장르별 베스트입니다. 각 장르별 베스트를 볼 수 있도록 data 속성 을 사용해 탭메뉴를 구현했으며 사용자의 점수는 몇점인지 얼마의 유저가 리뷰를 달았는지 좀 더 직관적으로 보일 수 있도록 제작했습니다.

SEARCH

모바일 사용자에 적합하게 검색페이지를 제작해 적절한 크기로 배치하였습니다.

COMMUNITY

사용자들이 자유롭게 웹소설에 대해 이야기를 나눌 수 있는 공간입니다. 소설네트워크의 주색상인 #23D2E2를 글쓰기 버튼에 배치해 모바일 웹 요소의 컨텐츠를 사용자가 좀 더 직관적으로 알 수 있도록 하였습니다
